#  Employee Attrition & Performance Analysis
**Industry Project | HR Analytics Pipeline**  
**Author:** Tanvi Sachindra Rahate  
**Institute:** Guru Nanak College of Arts, Science and Commerce  
**Project Dates:** 10-04-2026 to 21-04-2026  

##  Objective
End-to-end HR analytics workflow that identifies attrition drivers, flags high-risk cohorts, and delivers actionable retention strategies using Power BI + Python.

**Key Findings:**
-  Overall attrition: `16.1%` (benchmark: 10–12%)
-  Overtime employees leave at `~3x` the rate
-  Leavers earn `30% less` than stayers in identical roles
-  Highest-risk cohort: Sales Reps, Age 26–35, OT=Yes, Tenure <2 yrs
